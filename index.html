Brew Update Php Default Ke Versi 7 4

Mon, Dec 2, 2019 Setelah melakukan update dan upgrade brew, tiba-tiba CMS yang berbasiskan OctoberCMS gagal untuk berjalan sempurna. Belum sempat review untuk update code terbaru dari team OctoberCMS, akhirnya coba kita install php versi 7.3 kembali. brew install php@7.3 Kemudian, buat symbolic link untuk php73 agar bisa digunakan dengan command php73 di cli. cd /usr/local/bin/ ln -s php73 ../Cellar/php@7.3/7.3.12/bin/php untuk sementara menjalankan menggunakan php73 saja dulu. php73 artisan serve

Remote Fatal Unable to Create Thread Resource Temporarily Unavailable

Sat, Oct 26, 2019 Permasalahan pada saat git pull dari repository client yang menggunakan shared hosting CPanel dengan status remote: fatal: unable to create thread: Resource temporarily unavailable Hasil pencarian didapatkan hasil dari stackoverflow dengan menaikkan pack.threads di shared hosting milik client. git config --global pack.threads "5" Referensi : https://stackoverflow.com/questions/9905257/git-push-fatal-unable-to-create-thread-resource-temporarily-unavailable
